# InternStart C++ backend

Экспериментальный C++17 backend, который заменяет Flask-сервер самодостаточным HTTP-сервером поверх POSIX sockets и SQLite.

## Запуск

```bash
cd cpp_backend
make
PORT=5050 ./internstart_cpp
```

По умолчанию используются те же `.env` / `.env.local` и `DATABASE_URL`, что и у Flask-версии.

## Что уже перенесено

- Инициализация SQLite-схемы.
- Публичный каталог стажировок.
- Детальная страница стажировки.
- Список компаний и страница компании.
- Трекер подач.
- Админ-логин по `ADMIN_USERNAME` / `ADMIN_PASSWORD`.
- Админский CRUD для стажировок и компаний.
- Переключение публикации.
- Статическая раздача `cpp_backend/static/styles.css`.

## Что осталось следующим слоем миграции

- Полная визуальная совместимость с Jinja-шаблонами.
- Подробный редактор таймлайна подачи.
- Сетевой импорт из `Работа России`.
- OpenRouter-интеграция, которая в текущем Python-коде тоже отключена.
